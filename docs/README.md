# 效率平台技术栈

本文档介绍了在效率平台做后端开发过程中使用的技术栈，做后台开发要了解的技术，大家分别整理学习，并且将文档上传至此做交流分享，技术要有深度，不要停留在使用的层次，要更加深入的挖掘底层的内容，技术内容不只有这些，大家如果在工作学习中有想要分享的也可以在这里添加。

写文档是一个长期的过程，需要大家在后期不停的添加与修改，查漏补缺。大家之后在写完文档以后记得在这里添加一下链接。
